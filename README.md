# 📊 Superstore Online Sales Analysis (Power BI)

## 🔍 Project Overview
This project presents an end-to-end Business Intelligence solution to analyze the sales performance of a global retail superstore. The dashboard provides actionable insights into revenue trends, profitability, customer behavior, and regional performance using Power BI.

---

## 🎯 Objectives
- Analyze sales, profit, and order trends
- Identify top-performing categories and regions
- Evaluate customer behavior and return patterns
- Build an interactive dashboard for business decision-making

---

## 📌 Key Performance Indicators (KPIs)
- **Total Sales:** $1.57M  
- **Total Profit:** $175.26K  
- **Total Orders:** 5.9K  
- **YoY Growth:** 177.29%  
- **Total Customers:** 773  

---

## 📈 Key Insights

### 🛍️ Product Analysis
- **Office Supplies** leads in total orders and quantity sold  
- **Technology** category generates higher profit margins  
- Top sub-categories: Copiers and Accessories  

### 🌍 Regional Analysis
- **West Region** contributes highest sales (33.37%)  
- Followed by East and Central regions  
- Strong presence in the United States  

### 📅 Sales Trends
- Sales peak during **November and December**  
- Consistent growth observed across months  

### 🚚 Operations Insights
- **Standard Class** is the most used shipping mode  
- **Cash on Delivery (COD)** is widely preferred  

### 🔁 Return Analysis
- Low return rate with **94.94% successful deliveries**  

---

## 🛠️ Tech Stack
- **Power BI Desktop** – Dashboard & Visualization  
- **Power Query** – Data cleaning & transformation  
- **DAX** – Calculated measures (YoY, KPIs)  
- **Data Modeling** – Star schema relationships
